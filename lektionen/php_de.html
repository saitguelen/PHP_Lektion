🐘 PHP: Von den Grundlagen zum Profi

Serverseitige Skriptsprache

Lass uns Schritt für Schritt die Technologie erlernen, die das Rückgrat des Webs bildet.

🤔 Warum PHP?

🌐 Weite Verbreitung

~78% der Websites im Internet nutzen PHP. Giganten wie WordPress, Wikipedia und Facebook haben mit PHP begonnen.

🚀 Einfacher Einstieg

Die Syntax ähnelt C und ist leicht zu erlernen. Es hat eine riesige Community und eine umfangreiche Dokumentation.

🛠️ Großes Ökosystem

Mit modernen Frameworks wie Laravel, Symfony und Paketmanagern wie Composer ist die Entwicklung professioneller Anwendungen sehr einfach.

⚙️ Wie funktioniert es?

PHP ist eine serverseitige Sprache. Der Code wird auf dem Server ausgeführt und als reines HTML an den Browser gesendet.

👨‍🍳

Analogie: Restaurant. Sie (Benutzer) bestellen. Der Kellner (Browser) leitet die Bestellung an die Küche (Server) weiter. Der Koch (PHP) bereitet das Essen (dynamischer Inhalt) zu und der Kellner serviert Ihnen nur das Ergebnis (HTML-Seite).

💻 (Anfrage) → 🌐 → 🖥️[Apache + 🐘PHP] → 🌐 → 💻 (Antwort HTML)

📝 Grundlegende Syntax

PHP-Code wird zwischen <?php ... ?>-Tags geschrieben. Jeder Befehl endet mit einem ;.

<?php
  // Dies ist eine Kommentarzeile.
  echo "Hallo Sait!"; // Wird verwendet, um Text auf dem Bildschirm auszugeben.
?>
StrukturErklärung
echoGibt eine oder mehrere Zeichenketten auf dem Bildschirm aus.
// ...Wird für einzeilige Kommentare verwendet.
/* ... */Wird für mehrzeilige Kommentarblöcke verwendet.

📦 Variablen und Datentypen

Variablen sind Container zur Speicherung von Daten und beginnen mit einem $-Zeichen.

$name = "Sait";                  // String (Zeichenkette)
$alter = 30;                     // Integer (Ganzzahl)
$istStudent = true;              // Boolean (true/false)
$kurse = ["Java", "PHP"];        // Array (Feld)
$leererWert = null;              // NULL (Leer)

PHP ist eine "loosely typed" Sprache. Man muss den Typ einer Variable nicht im Voraus deklarieren.

🔍 Variablenprüfung: isset(), empty()

Dies sind entscheidende Funktionen zur Überprüfung des Zustands von Variablen, insbesondere bei der Verarbeitung von Formulardaten.

isset($var)

Prüft, ob eine Variable deklariert ist und ihr Wert nicht null ist. Gibt true zurück, wenn sie deklariert ist.

$a = "test"; isset($a); // true
$b = null; isset($b); // false
isset($c); // false (nicht deklariert)

empty($var)

Prüft, ob eine Variable als "leer" gilt. Werte wie "", 0, "0", null, false, [] gelten als leer.

$x = ""; empty($x); // true
$y = 0; empty($y); // true
$z = "a"; empty($z); // false

🚦 Kontrollstrukturen: if/else

Steuert den Programmfluss basierend auf bestimmten Bedingungen.

$note = 75;
if ($note >= 85) {
  echo "Sehr Gut";
} elseif ($note >= 70) {
  echo "Gut";
} else {
  echo "Bestanden";
}

🔁 Schleifen: for & foreach

Wird verwendet, um einen Codeblock zu wiederholen.

for-Schleife

for ($i = 1; $i <= 3; $i++) {
  echo "Zahl: $i <br>";
}

foreach-Schleife

$kurse = ["Java", "PHP"];
foreach ($kurse as $kurs) {
  echo $kurs . "<br>";
}

📊 Arrays

Ermöglicht das Speichern mehrerer Werte in einer einzigen Variablen.

Indizierte Arrays

$autos = ["BMW", "Mercedes", "Audi"];
echo $autos[0]; // Ausgabe: BMW

Assoziative Arrays

$person = [
  "name" => "Sait",
  "alter" => 30
];
echo $person["name"]; // Ausgabe: Sait

🔧 Häufige Array-Funktionen

$array = ["Apfel", "Birne"];

// Gibt die Anzahl der Elemente zurück
count($array); // 2

// Fügt ein Element zum Array hinzu
array_push($array, "Erdbeere");

// Entfernt das letzte Element des Arrays
array_pop($array);

// Sortiert das Array
sort($array);

// Wandelt ein Array in eine Zeichenkette um
$string = implode(", ", $array); // "Apfel, Birne"

// Wandelt eine Zeichenkette in ein Array um
$neuesArray = explode(", ", $string);

🧩 Funktionen

Wiederverwendbare Codeblöcke. "Einmal schreiben, überall verwenden."

// Funktionsdefinition
function begruesse($name = "Gast") {
  return "Hallo, " . $name . "!";
}

// Funktionsaufruf
echo begruesse("Sait"); // Hallo, Sait!
echo begruesse();      // Hallo, Gast!

📨 Formulardaten verarbeiten: GET & POST

PHP fängt Daten aus HTML-Formularen mit den superglobalen Arrays $_GET und $_POST ab.

// form.html
<form action="verarbeitung.php" method="post">
  <input type="text" name="benutzername">
  <button type="submit">Senden</button>
</form>

// verarbeitung.php
if (!empty($_POST["benutzername"])) {
    echo "Willkommen, " . htmlspecialchars($_POST["benutzername"]);
}

Wichtig: Verwende aus Sicherheitsgründen immer htmlspecialchars(), wenn du Benutzereingaben ausgibst!

🍪 Cookie- und Session-Management

Wird verwendet, um Benutzerinformationen und den Status (z. B. ob eingeloggt) zu verfolgen.

Cookie

Wird im Browser des Benutzers gespeichert. Ideal für Funktionen wie "Angemeldet bleiben".

// Erstellt ein Cookie, das 1 Stunde gültig ist
setcookie("benutzer", "Sait", time() + 3600);
// Cookie lesen
echo $_COOKIE["benutzer"];

Session

Wird auf dem Server gespeichert. Wird für sichere Daten (wie Benutzer-ID) verwendet. Sicherer.

session_start(); // Muss immer am Anfang stehen!
// Daten in die Session schreiben
$_SESSION["user_id"] = 123;
// Daten aus der Session lesen
echo $_SESSION["user_id"];

📅 Datums- und Zeitfunktionen

Wird verwendet, um Datum und Uhrzeit zu formatieren und zu bearbeiten.

// Zeigt das aktuelle Datum und die Uhrzeit
echo date('d.m.Y H:i:s'); // Bsp: 29.08.2025 14:22:38

// Zeitstempel erstellen
$morgen = mktime(0, 0, 0, date("m"), date("d")+1, date("Y"));
echo date('d.m.Y', $morgen);

// Modernerer Ansatz: DateTime-Objekt
$date = new DateTime();
$date->modify('+1 day');
echo $date->format('d.m.Y');

🧮 Mathematische Funktionen

Integrierte Funktionen für häufige mathematische Operationen.

// Erzeugt eine Zufallszahl
echo rand(1, 100);

🚗 OOP-Einführung

OOP ist ein Programmierparadigma, das Code um Objekte und Klassen herum organisiert.

🏠

Analogie: Eine Klasse ist der Bauplan für ein Haus. Ein Objekt ist ein echtes Haus, das nach diesem Plan gebaut wurde.

class Auto {
  public $farbe; // Eigenschaft (property)
  
  public function __construct($farbe) {
    $this->farbe = $farbe;
  }

  public function starten() { // Methode (method)
    return "Das Auto ist gestartet!";
  }
}

$bmw = new Auto("Schwarz"); // Objekt erstellen
echo $bmw->farbe; // "Schwarz"

🏛️ OOP-Grundkonzepte

Kapselung (Encapsulation)

Bündelt Daten und Methoden in einer Klasse und beschränkt den Zugriff von außen (public, private, protected).

Vererbung (Inheritance)

Eine Klasse erbt Eigenschaften und Methoden von einer anderen Klasse (extends).

Polymorphie (Polymorphism)

Verschiedene Klassen implementieren Methoden mit demselben Namen auf unterschiedliche Weise.

➕ include und require

Ermöglicht das Einbinden einer PHP-Datei in eine andere, um Ihren Code modularer zu gestalten.

include

Gibt eine Warnung (warning) aus, wenn die Datei nicht gefunden wird, und setzt die Ausführung fort.

// header.php
<h1>Willkommen auf meiner Seite</h1>

// index.php
include 'header.php';
echo "Inhalt der Hauptseite.";

require

Verursacht einen fatalen Fehler (fatal error), wenn die Datei nicht gefunden wird, und stoppt die Ausführung. Wird für kritische Dateien (z. B. DB-Verbindung) verwendet.

// db_config.php - Datenbankeinstellungen
require 'db_config.php';
// ... Datenbankoperationen

include_once und require_once stellen sicher, dass eine Datei nur einmal eingebunden wird.

🗄️ Datenbankverbindung: PDO

In modernem PHP wird PDO (PHP Data Objects) für Datenbankoperationen verwendet. Es ist sicher und flexibel.

try {
  $conn = new PDO("mysql:host=localhost;dbname=test", "root", "");
  $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

  // Sichere Methode gegen SQL-Injection
  $stmt = $conn->prepare("SELECT * FROM users WHERE id = :id");
  $stmt->execute(['id' => 123]);
  $user = $stmt->fetch();

} catch(PDOException $e) {
  echo "Verbindungsfehler: " . $e->getMessage();
}

📦 Composer - Der Dependency Manager

Ähnlich wie Maven in Java oder NPM in Node.js. Es verwaltet die Bibliotheken (Pakete), die Ihr Projekt benötigt.

🛒

Analogie: Sie erstellen eine Einkaufsliste (composer.json) mit Zutaten (Bibliotheken) für Ihr Rezept (Projekt). Der Befehl composer install holt alle Zutaten für Sie und bringt sie in Ihre Küche (vendor-Ordner).

✨ Modernes PHP: Namespaces & Autoloading

Entscheidend für die Code-Organisation in großen Projekten.

Namespaces

Verhindert Namenskonflikte zwischen Klassen. Man kann es sich wie eine virtuelle Ordnerstruktur vorstellen.

namespace App\Controllers;
class UserController { }

Autoloading

Dank Composer werden Klassen bei Bedarf automatisch geladen. Die Flut von require-Anweisungen entfällt.

use App\Controllers\UserController;
$controller = new UserController();

🏗️ Warum ein Framework verwenden?

Frameworks bieten eine vorgefertigte Struktur, Bibliotheken und Best Practices. Sie verhindern, dass wir das Rad neu erfinden.

Laravel

Bekannt für schnelle Entwicklung, elegante Syntax und Anfängerfreundlichkeit.

Symfony

Ideal für große und unternehmenskritische Projekte. Sehr flexibel durch seine modulare Struktur.

Als Fachinformatiker für Anwendungsentwicklung ist es für deine Karriere sehr wichtig, mindestens ein PHP-Framework gut zu beherrschen.
